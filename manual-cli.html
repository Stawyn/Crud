Manual - ProjectCloud CLI

Nesta página você pode encontrar informações sobre o ProjectCloud CLI

Menu Inicial

No menu inicial você vera varias opções como Visualizar, Inserir, Editar, Deletar e Exportar escolha a opção que você desejar.

Visualizar

Ao Visualizar aparecera todos os registros cadastrados em uma forma de tabela.

Inserir

Escolhendo inserir você deve preencher o formulario que aparecera na tela não esqueça que itens como ram, armazenamento, fonte, preço devera ser preenchido com numeros.

Editar

Escolha o id do seu registro para alterar seu registro ele mostrara os dados antigos para você ter uma noção de como fazer o novo registro

Deletar

Escolha o id do seu registro para deletar não ha como voltar então preste atenção ao deletar

Exportar

A ultima opção do menu principal é a exportar lá você tem duas opções CSV ou TXT dependendo de qual você escolher vira no formato que você escolheu

Licença

Para ter acesso ao sistema você precisará de uma licença porem para obter uma licença você precisa entrar em contato para obter uma.
